# 🐍 Snake & Ladder — Browser Game

A fully playable **Snake & Ladder** game built in the browser with no downloads required. Built as a fun hobby project using AI (Claude Code) — no prior game development experience needed!

## 🎮 Play Now

👉 **[snakeladder-one.vercel.app](https://snakeladder-one.vercel.app/)**

---

## ✨ Features

- **Two game modes** — Player vs Player (pass & play) or Player vs Computer
- **Realistic board** — SVG-rendered snakes and ladders with animations
- **Step-by-step movement** — tokens move one square at a time like a real board game
- **Sound effects** — dice rattle, snake hiss, ladder chime, win fanfare & background music
- **Post-game analytics** — stats, dice history chart, and a psychological profile card per player
- **Leaderboard** — tracks wins, win rate, snakes hit, ladders climbed across all games
- **Admin dashboard** — PIN-protected analytics panel with full game history and export
- **Keyboard support** — press `Space` to roll, `Ctrl+Space×2` for auto-roll mode
- **Mobile friendly** — triple tap the dice to activate auto-roll on touch screens
